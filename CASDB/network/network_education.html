This is a social network based on where scientists attended school. The nodes represent individual scientists and edges linking the scientists represent that both individuals attended the same college/university. Different colored links represent different institutions.
The 6 main institutions are represented by the following colors:
-Purple: 私立金陵大學
-Green: 國立中央大學
-Light-Blue: 國立浙江大學
-Orange: 國立四川大學
-Light-Gray: 國立西北農學院
-Red: Cornell University
